## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn
- Stacks Wallet

### Installation

1. Clone the repository
```bash
git clone https://github.com/Dayeneriss/aggrega-stacks.git
cd aggrega-stacks
```

2. Install dependencies
```bash
cd frontend
npm install
```

3. Run the development server
```bash
npm start
```
